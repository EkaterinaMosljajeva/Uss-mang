# Uss-mang
Без наушников игра не запускается!!
При запуске программы выдается меню из 3 функций.
1. Запускает игру. При выборе этого пункта программа запрашивает имя. При его вводе (от 3 символов) игра начинается. Есть ловушки (#)(Пока что не работают), еда, увеличение скорости в зависимости от очков.
2. Показывает топ всех игроков
3. Меняет цвет змеи (Если не выбрать, будет стандартно зеленый)
